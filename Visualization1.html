Dataset description:

Road Safety Authority Data.

This data set contains the data that is provided by Road Safety Authority (RSA) related to vehicle makes and models that were tested by the National Car Testing Service in 2016
Initial questions How do the total number of tests conducted relate to the most tested/manufactured vehicle?
Description:  This bar chart show the number of tests performed for different vehicles. Each bar on y-axis represents one distinct Vehicle, whereas the x-axis consists of the number of tests performed.
Insight: It is pretty evident from the visualization that the most number of tests were performed for Toyota. Which also makes us realize that Toyota was the most manufactured and tested car along these years, followed by Volkswagen, Ford and so on.
Design considerations I have used a bar chart as it is a very effective way to show two entities and display the required insights in a simpler way. It also helps us view key values at one glance itself. For example, we can tell the maximum tests were performed for Toyota whereas the least number of tests were performed for Jaguar. So it makes it easy to compare the number of tests for any two vehicleMakes in consideration.
Data filtering and transformation:  Only those Vehicles for which the number of tests performed were greater than 300 are shown.
