Dataset description:

Road Safety Authority Data.

This data set contains the data that is provided by Road Safety Authority (RSA) related to vehicle makes and models that were tested by the National Car Testing Service in 2016
Initial questions Out of all the tests performed by NCT, for all years of car manufactured, how many of them passed?
Description:  This bar chart shows the total tests that were conducted by NCT for every year along with the passed tests. Every bar corresponds to the total number of tests performed for a particular year wherein the x-axis represents the years and number of tests are represented on y-axis.
Insight: This visualizatiom shows that the tests performed in the early 90s were very less but this number rose starting from 95 and saw a peaked value for the year 2005, followed by 2008 and so on. Using this visualization we can compare the number of total tests performed. However, the data is a bit skewed and not uniform in the number of total tests so comparing the passed tests between any two years seems difficult, although ratios could be taken into consideration. We also see that the tests performed in the end were almost negligible (in the years 2014-15).
Design considerations I first visualized this using a line chart for both the entities(Total tests and passed ones), although it does show the peak points that a line chart shows, but it's difficult to compare the total tests with the passed one for a particular year. So I visualized it with a bar chat. A bar chart is the best way to show comparison between various competing values. Like we could compare the tests between various years easily with a bar chart. I used blue colour to represent the total tests conducted whereas Green colour to show the ones that have passed, as green is best used to represent positive/pass results.
Data filtering and transformation:  Only the data for the years in the range [1990, 2016] has been taken into consideration
